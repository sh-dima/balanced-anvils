How to Export the Mod Icon
==========================

Use Blockbench version 5.1.4.

Go to View > Advanced Screenshot and use the following settings:

* Angle Preset: Isometric Right (2:1)
* Resolution: 1024x1024
* Zoom / Focal Length: 92
* Anti-aliasing: Supersampling
* Show Edit Gizmos: False
* Shading: True
