Balanced Anvils
===============

Makes anvil penalties (experience cost, anvil usage increase) of certain anvil interactions more balanced.

Balanced Anvils is fully functional as a server-side mod, but can be installed on the client and used in singleplayer too.

Features
--------

The experience costs added by repairing and renaming are removed. The anvil usages will not increase for renaming/repairing items.

In other words, if the result does not have different enchantments compared to the first item, the interaction will be free. Otherwise, it will cost experience and increase the anvil uses.
